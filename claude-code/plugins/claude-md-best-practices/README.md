# CLAUDE.md Best Practices Plugin

Guide for writing and maintaining effective CLAUDE.md files.

## Components

- **1 skill** (`writing-claude-md`): Auto-activates when creating, editing, or reviewing CLAUDE.md files

## Skill

The `writing-claude-md` skill activates when:
- Creating or modifying CLAUDE.md files anywhere in the repo
- Claude makes a repeatable error that should become a rule

Based on practices from Boris Cherny (Claude Code creator), Thariq Shihab (Anthropic), and official Anthropic documentation.

## Usage

The skill activates automatically when working with CLAUDE.md files. No commands needed.
